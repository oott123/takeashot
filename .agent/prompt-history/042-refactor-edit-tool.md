将选择/移动工具拆成两个工具，图标请读取 references/tabler-icons/icons 这个目录下的 svg 文件并在 QML 中绘制：

1. 移动选区工具。
这是默认模式，和现在的“选择/移动工具”几乎相同，但用户不可以在该工具下选中标注。图标保持和现有工具一样不变。

2. 标注编辑工具。

在本工具下，用户只可以选择标注，并对标注进行移动、缩放、旋转等修改。
但不能对选区进行移动、修改等操作，就像其它标注工具一样。
如果用户鼠标下没有任何标注，光标为普通箭头。如果有标注，则为移动型。
用户选中标注后，可以拖动标注，缩放标注，删除标注等等。

图标选择 `edit.svg`。
