在**有选区状态**下，增加一个工具条。
1. 工具条的 UI 暂时先用一个宽 200 ，高 40 的矩形框代替。
2. 工具条需要智能选择位置。优先在选区外、右对齐、下方。如果下方没有空间，那就选择上方。
如果选区外没有空间，才可以放到选区里，也是遵循尽量放到右下方的原则。
3. 在选区变化的时候，工具条需要跟着移动。
4. 工具条永远显示在屏幕上，不要被选区窗口遮挡。
5. 工具条实现要放到一个单独的文件里。

---

工具条应该在什么状态下显示？
Answer: 只在有选区状态时显示

工具条应该包含哪些功能按钮？
Answer: 先不包含功能，只用一个框代替

工具条的定位策略是否需要调整？
Answer: 出现在选区外右下，没空间就选区内右下
